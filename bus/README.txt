Программа показывает, какой вид общественного транспорта ходит по городскому маршруту города Екатеринбурга.
Программа создана по причине отсутствия точного обозначения транспорта в маршрутах на онлайн картах.
Данные взяты с офицального портала Екатеринбурга: https://екатеринбург.рф/жителям/транспорт/перевозчики
